# Лабораторная работа №1 - Введение в Python

## Описание
Этот проект содержит решения 11 заданий по основам программирования на Python.

## Структура проекта
lab1-python-intro/

├── main.py # Главный модуль

├── src/ # Исходные модули с решениями

├── tests/ # Тесты для всех модулей

├── requirements.txt # Зависимости

└── README.md # Этот файл

## Установка и запуск
1. Клонируйте репозиторий:
  git clone <url-репозитория>
  cd lab1-python-intro

Установите зависимости:
pip install -r requirements.txt

Запустите основную программу:
python main.py

Запустите тесты:
pytest tests/ test_my_functions.py


Задания

00_distance.py - Расчет расстояний между городами

01_circle.py - Площадь круга и проверка точек

02_operations.py - Математические операции

03_favorite_movies.py - Работа со строками (фильмы)

04_my_family.py - Работа со списками (семья)

05_zoo.py - Операции со списками (зоопарк)

06_songs_list.py - Работа со списками и словарями (песни)

07_secret.py - Декодирование сообщения

08_garden.py - Операции с множествами

09_shopping.py - Анализ цен

10_store.py - Расчет стоимости товаров

Тестирование
Проект включает полный набор тестов для проверки корректности всех решений.
Для запуска тестов запустите cmd, введите python -m pytest tests/test_my_functions.py -v --tb=short --durations=0
